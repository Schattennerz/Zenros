# Zenros

Ein kleines Browser‑Projekt mit HTML, CSS und JavaScript.

## Inhalt
- Einstieg über [index.html](index.html)
- Weitere Startseite: [startscreen.html](startscreen.html)
- JavaScript‑Module für Spiellogik und Inhalte

## Projektstruktur
- [index.html](index.html) – Hauptseite
- [startscreen.html](startscreen.html) – Startbildschirm
- [style.css](style.css) – Styles
- [player.js](player.js) – Spielerlogik
- [src/](src/) – Ausrüstung, Materialien, Zauber
- [enemis/](enemis/) – Gegner
- [folk/](folk/) – NPCs/Völker
- [router/](router/) – Routing/Navigation

## Starten
- Öffne [index.html](index.html) im Browser.
- Optional: Mit VS Code „Live Server“ starten.

## Entwicklung
- Änderungen an HTML/CSS/JS speichern
- Seite im Browser neu laden
